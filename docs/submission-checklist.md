## Before You Submit

1. Open the live app and test one full Hindi tutoring flow.
2. Record a 2 minute demo video using the script in [demo-script.md](/Volumes/CodeDrive/OpenSourceGithub/nanhipathshala/docs/demo-script.md).
3. Upload the GitHub repository link.
4. Upload the Cloud Run web app link.
5. Upload the deck or export it to PDF if the form asks for slides.
6. Paste the project description from [submission-copy.md](/Volumes/CodeDrive/OpenSourceGithub/nanhipathshala/docs/submission-copy.md).
7. Be ready to explain why this is an ADK agent, not just a chatbot.
