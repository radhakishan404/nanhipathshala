# Submission Copy

## Project Title

NanhiPathshala

## One-Line Pitch

NanhiPathshala is a Hindi-first AI tutoring agent for Indian children and mothers, built with Google ADK, Gemini on Vertex AI, and Cloud Run.

## Problem Statement

Many Indian mothers want to help their children study at home, but most digital learning tools are English-first, expensive, or too complex for young children. Children often need simple Hindi explanations, patient repetition, and emotional encouragement. Mothers need a quick and practical summary of what the child learned and where help is still needed.

## Solution

NanhiPathshala gives families a gentle AI tutor that teaches in simple Hindi, retries when the child says they did not understand, asks short practice questions, and generates a parent-friendly summary for the mother.

## What We Built

- Hindi-first tutoring flow
- Mobile-friendly web interface
- ADK-based teaching agent
- Quiz generation tool
- Reteach tool for confused learners
- Parent summary tool
- Cloud Run deployment on Google Cloud

## Target Users

- Mothers supporting children in classes 1 to 5
- Young learners who are more comfortable in Hindi than English
- Families looking for a low-cost home learning companion

## Why This Matters

The project reduces learning stress at home by giving children a patient tutor and giving mothers a simple summary instead of a complex dashboard.

## Google Technology Used

- Google Agent Development Kit (ADK)
- Gemini 2.5 Flash
- Vertex AI
- Cloud Run
- Cloud Build

## Why This Fits Track 1

NanhiPathshala directly matches Track 1 because it is built and deployed as an AI agent using Gemini, ADK, and Cloud Run.

## Future Scope

- Native Gemini Live API audio conversations
- Regional language support beyond Hindi
- Simple progress history for repeated lessons
- School-aligned topic packs by class level
