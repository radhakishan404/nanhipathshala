# Project Brief

## Name

NanhiPathshala

## Problem

Many Indian mothers want to help their children learn at home, but most digital learning tools are expensive, English-first, or not designed for young children who need patient, emotionally supportive guidance.

## Solution

NanhiPathshala is a Hindi-first AI tutor that explains concepts simply, speaks with an encouraging tone, adapts when a child does not understand, and gives mothers a short summary of progress.

## Track Fit

Track 1: Build and deploy AI agents using Gemini, ADK, and Cloud Run
