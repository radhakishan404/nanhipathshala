# Demo Script

## Demo Goal

Show that NanhiPathshala is a Hindi-first tutoring agent for Indian families and not a generic chatbot.

## 2 Minute Flow

1. Open the live web app.
2. Show the Hindi-first interface and the large simple controls.
3. Start a lesson for a class 2 or class 3 child.
4. Ask a simple topic question such as fractions or plants.
5. Show the agent replying in simple Hindi.
6. Say or type `samajh nahi aaya`.
7. Show the agent reteaching the same topic more simply.
8. Trigger a short practice quiz.
9. Open the parent summary card and explain how the mother can see what the child learned.
10. End by showing the architecture slide or submission note that it runs on Google ADK, Gemini, Vertex AI, and Cloud Run.

## Judge Closing Line

NanhiPathshala helps Indian mothers and children with a low-cost Hindi AI tutor that is patient, practical, and deployable today on Google Cloud.
